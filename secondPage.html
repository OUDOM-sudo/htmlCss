Essential CSS Code You Should Know

CSS (Cascading Style Sheets) is used to style HTML elements. Below are key CSS properties and concepts every beginner should understand.

Common CSS Properties

  • color: Sets the text color, e.g., color: #333; for dark gray text.
  • font-size: Defines the size of text, e.g., font-size: 16px;.
  • margin: Controls the space outside an element, e.g., margin: 10px;.
  • padding: Controls the space inside an element, e.g., padding: 1em;.
  • background-color: Sets the background color, e.g., background-color: #f4f4f4;.
  • display: Defines how an element is displayed, e.g., display: flex; for flexible layouts.
  • border: Adds a border around an element, e.g., border: 1px solid black;.
  • text-align: Aligns text, e.g., text-align: center;.
  • width and height: Set the dimensions of an element, e.g., width: 100%;.
  • box-sizing: Controls how width and height are calculated, e.g., box-sizing: border-box;.
